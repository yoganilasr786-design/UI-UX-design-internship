# Task 2 – Email Template UI Design (BookMart)

## Objective
To design a clean and engaging promotional email template for an online bookstore to showcase new arrivals and offers.

## Brand
BookMart – Online Book Store

## Email Type
- Promotional Email
- Newsletter

## Tools Used
- Figma

## Design Overview
This email template promotes BookMart’s latest book arrivals and highlights key benefits to encourage users to explore and purchase books.

## Sections Included
- Brand Logo (BookMart)
- Header with navigation link (Go to Account)
- Hero Section with headline and book image
- Promotional message for new arrivals
- Primary CTA (Explore Now)
- Feature Highlights:
  - Wide Book Collection
  - Fast Delivery
  - Affordable Prices
  - Trusted by Readers
- Social Media Icons
- Footer with unsubscribe option

## Call To Action (CTA)
- Go to Account
- Explore Now
- Unsubscribe

## Design Style
- Soft pastel background
- Clear typography
- Strong CTA button for conversions
- Minimal and user-friendly layout
